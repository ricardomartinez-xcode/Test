# LifeLab · Simulador de Vida

Juego web estático tipo simulador de vida, creado con **HTML, CSS y JavaScript puro**.

> Proyecto original inspirado en mecánicas de simulación social. No usa assets, marcas ni código de terceros.

## Cómo jugar

Cuida a tu Sim y equilibra sus necesidades:

- Hambre
- Energía
- Diversión
- Social
- Higiene
- Casa

También puedes trabajar, estudiar, limpiar, decorar y completar metas. El progreso se guarda automáticamente en `localStorage`.

## Ejecutar localmente

No requiere instalación.

```bash
python3 -m http.server 5173
```

Abre:

```text
http://localhost:5173
```

También puedes abrir `index.html` directamente en el navegador.

## Deploy

Es un sitio estático. Puede publicarse en GitHub Pages, Vercel o Netlify.
